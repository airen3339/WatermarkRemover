# WatermarkRemover 水印去除工具 v1.5
图片去水印工具|批量图片水印去除工具|去除图片水印工具
## 产品概述

WatermarkRemover 是一款专业的图片水印去除工具，能够智能识别并去除图片中的各类水印（文字水印、Logo水印、半透明水印等），同时支持图像修复（Inpaint）功能，广泛应用于图片处理、素材整理、文档优化等场景。

## 核心功能

### 1. 水印去除模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **自动检测** | 智能识别水印区域 | 通用水印 |
| **浅色水印** | 去除白色/浅色水印 | 浅色文字水印 |
| **深色水印** | 去除黑色/深色水印 | 深色Logo水印 |
| **彩色水印** | 去除指定颜色水印 | 彩色Logo |
| **半透明水印** | 去除半透明叠加水印 | 半透明Logo |
| **自定义颜色** | 指定水印颜色去除 | 特定颜色水印 |
| **边缘采样** | 从边缘采样修复 | 角落水印 |
| **智能背景填充** | 统计背景色智能填充 | 纯色背景水印 |
| **Inpaint修复** | 智能图像修复 | 任意区域修复 |
| **透明处理** | 将水印颜色转透明 | PNG透明化 |

### 2. 图像修复 (Inpaint)
- **Fast Marching 算法**：基于距离场的快速修复
- **Telea 算法**：基于流体力学的平滑修复
- **区域选择**：手动框选需要修复的区域
- **智能填充**：根据周围像素智能填充

### 3. 智能背景填充 (新功能)
类似微软照片查看器的 AI 擦除功能，自动识别背景色并填充水印区域。

- **边缘统计**：统计选区四周边缘的像素颜色
- **背景色识别**：找出出现频率最高的颜色作为背景色
- **智能替换**：将选区内与背景色差异超过阈值的像素替换为背景色
- **平滑处理**：进行轻微高斯模糊以平滑边缘
- **参数调节**：
  - 阈值：控制颜色差异容差，值越大替换越多
  - 强度：控制替换程度，100%完全替换，更低值保留部分原色
  - 半径：边缘采样宽度和模糊半径
- **适用场景**：纯色或近似纯色背景上的水印

### 4. 倾斜水印处理（旋转角度功能）
处理倾斜的水印，无需预先旋转整张图片。

- **工作原理**：将选区内容旋转到指定角度后处理，再旋转回原角度合并
- **角度范围**：-45° 到 +45°
- **正值**：顺时针旋转
- **负值**：逆时针旋转
- **适用场景**：
  - 斜体文字水印
  - 倾斜的Logo水印
  - 带角度的签名水印
- **使用方法**：
  1. 框选倾斜的水印区域
  2. 拖动"旋转角度"滑块，使水印在脑中"旋转到水平"
  3. 选择合适的去水印模式
  4. 点击"应用并预览"

### 5. 批量处理
- **文件夹批处理**：支持整个文件夹批量去水印
- **格式保持**：保持原图格式输出
- **子文件夹处理**：可选择是否处理子文件夹
- **进度和日志**：实时显示处理进度和详细日志

### 6. 图片格式支持
- PNG（支持透明通道）
- JPG/JPEG
- BMP
- GIF

### 7. 方案管理
支持保存和加载处理参数方案，方便批量处理相同类型水印的图片。

**保存的参数包括：**
- 阈值、半径、强度、旋转角度
- 水印类型、对齐方式
- 多次迭代选项
- 自定义颜色列表和容差
- 选区位置信息（按对齐方式保存相对位置）

**使用方法：**
1. 调整好参数和选区后，点击"保存"按钮保存方案
2. 打开新图片后，选择方案点击"应用方案"
3. 选区会根据对齐方式自动适配新图片尺寸

## 使用场景

1. **图片素材处理**：去除下载图片的水印
2. **文档扫描优化**：去除扫描件上的印章、水印
3. **照片修复**：去除照片中的日期戳、相机标识
4. **设计素材**：清理素材库中的水印图片
